# Memory Protocol

## Search Memory

Use Engram first when:

- a bot or strategy may already exist
- the user references previous work
- you need prior architectural context

Suggested flow:
1. `engram_mem_context`
2. `engram_mem_search`
3. `engram_mem_get_observation`

## Save Memory

Save decisions, discoveries, bugs, conventions and summaries.

Important topics:
- strategy rationale
- backtest findings
- execution constraints
- risk rules
- architecture decisions
- review outcomes
